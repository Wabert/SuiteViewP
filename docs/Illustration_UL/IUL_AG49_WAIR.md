# IUL Crediting — AG49 Regimes, Asset Charge, and WAIR

Extracted from **RERUN (v20.0).xlsm** on 2026-07-08 via
`tools/extract_calcengine.py`. This is the reference for the engine-side
implementation of the three IUL-specific calculations:

1. **AG49 regime differences** — gate multiplier crediting / asset charge and
   select the variable-loan credit spread by regime.
2. **WAIR** — Weighted Average Interest Rate (the TAV block).
3. **Blended rate** — the simpler method; already implemented end-to-end
   (`models/index_strategies.compute_blended_rates` → effective blend →
   `current_interest_rate`).

## AG49 regimes (Rates_Control CR78:CS83, CO78:CP81)

| Index | Regime | Issue dates from |
|---|---|---|
| 1 | Prior to AG49 | 1900-01-01 |
| 2 | AG49 | 2015-09-01 |
| 3 | AG49A | 2020-11-25 |
| 4 | AG49B | 2023-05-01 |

- Date tier: `CS77 = IFERROR(MATCH(sINPUT_Issue_Date, CS79:CS87), 1)`
- **Applicable index (CP79):** `sAG49Index = MAX(2, tier)` — RERUN always
  floors at the original AG49 rules, even for pre-2015 policies.
- **Loan credit spread (CP80):** `= CHOOSE(sAG49Index, 0, 0.01, 0.005, 0.005)`
- `blnDisplayNominalInterestRate (CP81) = AND(sAG49Index<=2, allocIP>0 OR allocIR>0)`

App semantics: unchecked "Use Policy AG49 Regime" → current regime (index 4,
AG49B); checked → `MAX(2, issue-date tier)` (exactly RERUN CP79).

## Asset charge (CalcEngine SS:SV → monthly deduction)

Applies only to the IP/IR multiplier strategies under index ≤ 2:

```
SS  IP asset fee        = sIP_AssetFee            (0.0215)
ST  IR asset fee        = sIR_AssetFee            (0.0415)
SU  vBlendedAssetRate   = IF(sAG49Index <= 2, SS*allocIP + ST*allocIR, 0)
SV  vAssetCharge        = IF(sAG49Index <= 2, MAX(0, SU/12 * (OO - MS - MT)), 0)
      OO = mAV (AV before charge), MS/MT = regular loan principal/interest
      after loan repay — i.e. the charge base is the unloaned AV.
SX  vAV_AfterCharge     = OO - SW - IF(H, CHOOSE(sAG49Index, SV, SV, 0, 0))
      → the charge is deducted from AV only under regimes 1–2.
```

`compute_blended_rates` already returns `asset_charge_rate` (= SU); the engine
needs to deduct SV each month alongside the monthly deduction.

## WAIR (CalcEngine US..VL — the TAV block)

> RERUN's own header: "The TAV is ONLY used to calculate the Weighted Average
> Interest rate (WAIR). The TAV is a simplified projection of how much account
> value will be in the policy for the next year. It considers premiums (limited
> by 7702), loan repayments, loan balance, and Monthly deductions (from the
> SWAM — Sweep Account Minimum)."

Key rates: `UK = declared (fixed/sweep) rate + policy bonus`,
`UO = blended index rate (PolicyRates!CH4)`, `UP = UO + bonus`,
`sRates_LNCRD = loan credit rate`.

One-year TAV projection (computed on each anniversary row):

```
US  begin AV            = IF(beginOfYear, KW - vForceOut, 0)    KW = Prem-WD accum
UT  amount in 7-pay     = IF(beginOfYear, LE, 0)
UU  annualized sched prem = vPlannedPremium * LT (payments/yr)
UV  forecasted prem     = UU + vLumpsum
UW/UX  begin regular loan principal/interest (LX/LY)
UY/UZ  begin variable loan principal/interest (MB/MC)
VA  RgLn w interest     = UW*(1 + sRates_LNCRG*((13-month)/12)) + UX
VB  VblLn w interest    = UY*(1 + MAX(vblRate, UG-0.01)*((13-month)/12)) + UZ
VC  loan repayment      = IF(ApplyPremToLoan, MIN(UV, VA+VB), 0)
VD  prem less repay     = UV - VC
VE  prem capped (7702)  = MAX(0, IF(CVAT, VD, MIN(VD, NK)))     NK = guideline room
VF  TAV                 = IF(beginOfYear, BO + VE*(1 - premium load), 0)
VG  TAV display         = MAX(0, VF)
VH  SWAM                = IF(valuationDate, sINPUT_SWAM, SW*12)  SW = monthly deduction
```

WAIR itself — weight the sweep-minimum slice at the declared rate, the loaned
slice at the loan credit rate, and the remaining (indexed) slice at the blend:

```
VI  WAIR_on_ValuationDate = IF(AV<=0, 0,
      ( MIN(SWAM, AV)*UK
      + FixedLnPrincipal*sRates_LNCRD
      + (AV - (FixedLnPrincipal+FixedLnAccrued) - MIN(SWAM, AV))*UP ) / AV)

VJ  WAIR_on_Anniversary = IF(beginOfYear, IF(VG<=0, 0,
      ( MIN(VH, VF)*UK + TY*sRates_LNCRD
      + (VF - (TY + MT) - MIN(VH, VF))*UP ) / VF), VJ_prev)
      TY = total regular loan principal, MT = regular loan interest

VK  vWAIR = IF(assumptionCode=3, MIN(WAIR, UK), WAIR)   (guaranteed run caps at declared)
VL  WAIR interest = MAX(0, TF * ((1+VK)^(UD/365) - 1))  TF = vAV, UD = days
```

When `iul_wair_crediting` is on, VL replaces the plain blended-rate interest
credit; the WAIR is recomputed once per policy year (anniversary) and held.

## Variable-loan credit spread (CalcEngine VV)

```
VV  vbl loan accrued = MX + VU * IF(sAG49Index > 1,
        MAX(sINPUT_Variable_Loan_Rate, UO - sAG49LoanCreditSpread),
        sINPUT_Variable_Loan_Rate) * UD/365
```

The spread by index is the CP80 CHOOSE list above (`loan_credit_spread_by_index`
in the JSON; `loan_credit_spread_for_index()` in the model).
