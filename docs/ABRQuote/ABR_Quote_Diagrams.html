Python Conversion Plan • SuiteView Integration • February 2026

1. Calculation Flow

1. Policy Data

  • DB2 via PolView PolicyInformation
  • Policy #, Issue Age, Sex/Class
  • Face Amount, State, Plan Code
  • Table Rating, Flat Extra
  • Billing Mode, Paid-to Date
→

2. Term Premium Calc

  • Build lookup KEY
  • INDEX/MATCH into BaseTermPremiumRates
  • Add table rating (×0.25)
  • Add flat extra
  • Apply modal factor + $60 fee
→

3. Mortality Engine

  • 2008 VBT Select lookup (MN/FN/MS/FS)
  • Mortality improvement
  • Table rating multiplier
  • Flat extra additive
  • UDD monthly conversion
↓

4. Medical Assessment

  • 5yr / 10yr Survival Rates
  • Life Expectancy
  • Goal Seek → Table Rating (scipy)
  • Goal Seek → Flat Extra (scipy)
  • Assessment Index (1–7)
→

5. APV Engine

  • Monthly loop (~1,460 months)
  • PVFB = Σ(DB × v × tp'x × q'x) × adj
  • PVFP = Σ(Prem × v × tp'x)
  • Disc = Face − (PVFB − PVFP)
  • FL+Terminal → PVFP = 0
→

6. Output

  • Accel Benefit = Face − Disc − Fee
  • Fee: $250 (or $100 if FL)
  • Full + Max Partial sheets
  • Premium before/after
  • Export to Excel workbook

Key Formulas

Actuarial Discount = ROUND( Face − (PVFB − PVFP), 2 )

UDD Monthly qx = (annual_qx / 12) / (1 − (month_in_year − 1) × (annual_qx / 12))

Continuous Mort Adjustment = monthly_rate / ln(1 + monthly_rate)

Accelerated Benefit = Eligible_DB − Actuarial_Discount − Admin_Fee

Partial Discount = (Partial_Eligible / Full_Eligible) × Full_Discount

3. Module Structure

suiteview/
├── abrquote/
│   ├── __init__.py — Package init, launch_abrquote(), version
│   ├── main.py — Factory: create_abrquote_window()
│   │
│   ├── models/
│   │   ├── abr_constants.py — Bands, modal factors, plan codes, fees (~80 lines)
│   │   ├── abr_data.py — Dataclasses: ABRPolicyData, MedicalAssessment, etc. (~120 lines)
│   │   ├── vbt_2008.py — Embedded 2008 VBT mortality table + lookup (~500 lines)
│   │   └── term_rates.py — SQLite access: term rates + interest rates (~200 lines)
│   │
│   ├── core/
│   │   ├── mortality_engine.py — VBT lookup → improve → table → flat → UDD (~250 lines)
│   │   ├── apv_engine.py — Monthly APV loop: PVFB, PVFP, discount (~300 lines)
│   │   ├── premium_calc.py — Rate lookup, annual/modal premium (~150 lines)
│   │   └── goal_seek.py — scipy.optimize wrappers (~200 lines)
│   │
│   └── ui/
│      ├── abr_window.py — Main wizard window (FramelessWindowBase) (~500 lines)
│      ├── policy_panel.py — Policy info display panel (~200 lines)
│      ├── assessment_panel.py — Medical assessment form (~300 lines)
│      ├── results_panel.py — Quote results + export (~350 lines)
│      └── abr_styles.py — Teal & gold theme constants (~60 lines)
│
scripts/
└── run_abrquote.py — Standalone CLI launcher (~40 lines)

Total: ~3,300 estimated lines across 16 files
